# 快速开始

面向 Windows + PowerShell + Conda（项目目录：`D:\Cursor_project\SC2_RL`）。

## 1. 激活环境

```bash
conda activate sc2_rl
cd D:\Cursor_project\SC2_RL
```

## 2. 验证环境（推荐先跑）

无渲染（更快）：

```bash
python test_sc2_env.py --norender
```

有渲染（确认画面正常）：

```bash
python test_sc2_env.py
```

## 3. 跑一个基线 Agent（随机）

```bash
python simple_agent.py --episodes=10 --norender --max_steps=200
```

## 4. Q-Learning：训练与评估（MoveToBeacon）

### 4.1 训练

```bash
python mini_games_experiment/MoveToBeacon_1_Qlearning.py --episodes=500 --norender
```

训练完成后会输出并保存：
- `models/qlearning_movetobeacon.pkl`：Q 表
- `models/qlearning_movetobeacon_stats.pkl`：训练曲线数据

### 4.2 评估（只跑策略，不学习）

无渲染：

```bash
python mini_games_experiment/evaluate_qlearning.py --episodes=50 --norender --model_path=models/qlearning_movetobeacon.pkl
```

有渲染（观察行为）：

```bash
python mini_games_experiment/evaluate_qlearning.py --episodes=10 --render --model_path=models/qlearning_movetobeacon.pkl
```

## 5. 常用命令

查看可用地图：

```bash
python -m pysc2.bin.map_list
```

重新安装依赖：

```bash
pip install -r requirements.txt
```
